--------------

BIN28XP converts raw binary files that contain assembled
Z80-Code into the 8XP Format for use with TI-83+ calculators.

Syntax:

bin28xp [options] file

Options:

-c <comment>				Sets the comment, included in the 8XP File (<=42 Bytes).
					Default is "BIN28XP Converter by Joris Roos".
--nocomment				Sets the comment to an empty string.
--protected|--unprotected		If protected, the program can not be edited in the PRGM-Menu.
					Default is protected.
-n					Sets the name of the program that is showed in the PRGM-Menu (<=8 Bytes).
					Default is file name.
-q					Causes BIN28XP not to output any information on the screen describing
					the progress (quiet).
